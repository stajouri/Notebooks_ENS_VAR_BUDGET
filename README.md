A repository containing the coding material of the analyses related to the manuscript Tajouri et al., 2026.

# Content
The Computing_analyses repertory contains all the notebooks generating the data used to plot the figures of the manuscript.
The Plotting_figures repertory contains the notebooks to plot the figures of the main article and of the supplementary material. 


# Reference
Tajouri S., W.Llowel, F. Sevellec, Q. Jamet. A.Hochet (2026) : Impact of fully time-varying freshwater fluxes from Greenland and rivers on externally-forced and internal sea level variability. Journal of Geophysical Research
